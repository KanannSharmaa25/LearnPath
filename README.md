LearnPath

AI-Powered Personalized Learning Platform

By Kanan Sharma

Overview

LearnPath is a comprehensive learning management system that uses psychology-driven assessments and AI to create personalized learning paths. It analyzes your learning style, current skills, and goals to generate a customized roadmap for your tech career journey.

Features

🧠 Psychology Assessment

Understand how you learn best through comprehensive psychology-based questions covering learning style, interests, stress tolerance, and work preferences.

📝 Knowledge Quizzes

Test your skills across 20+ topics with 200+ expert-curated questions. Topics include Python, JavaScript, React, Machine Learning, SQL, and more.

🛤️ AI Learning Path

Get a personalized roadmap generated by AI based on your psychology profile, quiz results, and career goals. Choose between AI-generated, custom, or combined paths.

📚 Curated Courses

Access expert-selected courses matched to your unique learning needs and goals.

💬 AI Coach

Get 24/7 guidance and support from your personal AI learning assistant.

🔮 Smart Predictions

AI-powered timeline predictions to help you understand your learning journey and expected milestones.

📈 Progress Tracking

Track your learning journey with detailed analytics, learning logs, and visual progress indicators.

Tech Stack

Frontend: React, CSS3 (with animations)
Styling: Custom purple theme (#8b5cf6 primary)
State Management: React Context API
Build Tool: Create React App
Getting Started

Prerequisites

Node.js 16+
npm or yarn
Installation

# Clone the repository
git clone https://github.com/KanannSharmaa25/learnpath.git

# Navigate to the project
cd learnpath/learnpath-react

# Install dependencies
npm install

# Start the development server
npm start
The app will be available at http://localhost:3000

Build for Production

npm run build
